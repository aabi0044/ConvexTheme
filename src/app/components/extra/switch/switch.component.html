Switch

Checked

Use checked attribute Or [checked]="false", for default switch.

Reverse

Use reverse attribute to make it work like reverse switch.

Two way binding

For Two way binding switch, use [(ngModel)]="enable".

{{enable}}

Change

On change event of switch set counter using (change)="onChange()".

change count {{count}}

Disabled

Use disabled attribute Or [disabled]="true", for disabled switch.

Size

Use size="small" for small switch and size="large", for large switch.

color

Use color attribute to make switch colorful.

switchColor

Use switchColor attribute to make switch toggle colorful.
